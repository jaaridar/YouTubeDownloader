# YouTube Downloader Pro 🎬

A minimalist, high-performance Windows app for downloading YouTube videos in up to 8K Ultra HD.

## 🚀 Zero-Setup Design
**Just download `YouTubeDownloader.exe` and run it.** 
- The app automatically extracts and configures its own high-performance engine (`yt-dlp` and `ffmpeg`) into a local `bin` folder on the first run. 
- No manual downloads or complex installations are required.

## ✨ Key Features
- **Ultra Quality**: Supports 8K, 4K, 2K, and 1080p Full HD.
- **Visual Trimming**: Dual-slider range selector with live duration bubble.
- **Smart Hit-Testing**: Easily grab sliders even when they overlap.
- **Action Bar**: Instant "Play Video" and "Open Folder" buttons after download.
- **Pro Audio**: Extract high-quality MP3s with a single click.

## 🎯 How to Use
1. **Paste URL**: The app auto-fetches title, duration, and thumbnail.
2. **Select Quality**: Default is "Best Available".
3. **Optional Trim**: Toggle "Download Specific Section" and drag the sliders.
4. **Download**: One click and you're done.

## 🔧 Troubleshooting
- **Age Restricted**: If a video fails to fetch info, it might be age-restricted or private.
- **Engine Update**: Click "Update Engine" in the top-right to get the latest yt-dlp version if YouTube changes break the downloader.
